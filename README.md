🍽️ About the Project

The Golden Spoon is a modern, responsive restaurant website developed using HTML, CSS, and JavaScript. The project is designed to provide an elegant online presence for a restaurant, allowing visitors to explore the menu, learn about the restaurant, make reservations, and get in touch through a clean and user-friendly interface.

✨ Features
Responsive design for desktop, tablet, and mobile devices
Modern and attractive user interface
Home page with restaurant introduction
About Us section
Food Menu
Gallery
Reservation page
Contact page
Terms & Conditions
Privacy Policy
Support page
Smooth navigation
